# 安装并运行Juka (控制台)

Juka 可以在许多平台上运行，包括：

* 窗口
* macOS
* Linux
* FreeBSD
* Web browsers

## Installing Juka

若要安装 Juka，请遵循以下步骤：

1. 访问 [Juka 网站](https://jukalang.com/)。
2. 点击 **下载** 按钮。
3. 选择您想要安装 Juka 的平台。
4. 按照屏幕上的说明安装 Juka。
