# 条件

## 什么是条件？

一个条件是可以评估为真实或虚假的陈述。 使用了控制程序执行流程的条件。

## 语法

条件的语法如下：

```jsx
如果 <condition>


<statements>
}
否则
*
<statements>
}
```

例如，如果变量总和的值等于5，则以下代码将打印“正确!”，以及“不正确!”等于：


```jsx
sum = 4+1
if sum == 5{
    "Correct! 4+1 = 5"
}
else
{
    "Incorrect!"
} 4+1 = 5"
}
else
{
    "Incorrect!"
}
```
