# 字符串

## 什么是字符串？

字符串是一个字符串。 在编程中，字符串常用来表示文本。

## 定义字符串

字符串使用引号定义。 例如，下面的代码定义了一个名为 `动物` 的字符串，并将其设置为 "cat"的值：

```jsx
var animal = "cat"
```

## 组合字符串和数字

每当您将一个字符串添加到一个数字(或相反的数字)，最终结果将是一个字符串。 数字被默示转换为字符串。 例如，下列代码将打印字符串"123abc"：

```jsx
print("123" + "abc")
```
