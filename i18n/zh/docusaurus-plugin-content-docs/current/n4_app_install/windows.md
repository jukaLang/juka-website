# Windows JukaApp
Juka Windows 应用程序是学习和使用 Juka 编程语言的强大工具。 使用 Juka 应用程序，您可以：

* 在您的 Windows 设备上写入、编译和运行 Juka 代码。
* 通过交互式教程和练习学习Juka。
* 使用 Juka 语法和语义学获得帮助。
* 与他人分享您的 Juka 代码。

Juka Windows 应用程序在 [Juka 网站](https://jukalang.com/app/) 上是免费的。 要下载应用，只需访问网站，然后点击 **下载** 按钮。
